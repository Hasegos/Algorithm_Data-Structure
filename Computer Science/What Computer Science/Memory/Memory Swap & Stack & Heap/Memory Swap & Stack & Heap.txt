메모리 구조

힙 영역 : malloc를 통해 할당 받는 곳
스택 영역 : 함수 호출시 지역변수, 매개변수 할당
| 코드 영역 |
|데이터 영역|
|  힙 영역  |
|    ↓     | 
|           | 
|           |
|          |
|           |
|    ↑     |
| 스택 영역 | 
|  noswap() |  a - > 2, b  - > 1, tmp -> 1
|   main()  | : 스택 프레임 x -> 1, y -> 2

교환 -> 스택 영역이라 호출이 끝나면 사라짐 그래서 교환이 안됨
해결법 -> 참조로 넘기면됨, 반환값으로 주던가
noswap(int a, int b){
    int tmp = a;
    a = b;
    b = tmp
} 

&로 각각의 주소를 받은뒤 주소끼리 변경
swap(int *a, int *b){
    int tmp = *a;
    *a = *b;
    *b = tmp
}
